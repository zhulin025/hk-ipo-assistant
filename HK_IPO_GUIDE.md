## 基础概念

### 什么是港股打新？

港股打新是指申购即将在港交所上市的新股，并在上市后卖出获利的投资方式。

**核心逻辑**：
- 新股上市初期通常有溢价
- 通过申购获得原始股
- 上市后卖出赚取差价

### 关键术语

#### 基础术语

| 术语 | 解释 | 示例 |
|------|------|------|
| **一手** | 最小认购单位 | 100 股/手、500 股/手 |
| **入场费** | 认购一手所需金额 | 发行价 × 手数 + 费用 |
| **中签率** | 成功获配的概率 | 10% = 10 个人中 1 个 |
| **超募倍数** | 认购金额/发行金额 | 100 倍 = 超额 100 倍认购 |
| **暗盘** | 上市前一日的场外交易 | 富途/老虎内部交易 |
| **孖展** | 融资认购（杠杆） | 10 倍孖展 = 1 万认 10 万 |
| **基石投资者** | 上市前确定的大机构 | 腾讯、高瓴等 |
| **绿鞋机制** | 超额配售选择权 | 稳定股价机制 |

#### 进阶术语

| 术语 | 解释 | 重要性 |
|------|------|--------|
| **甲组** | 认购金额 <500 万港币 | 散户通道，一手必中 |
| **乙组** | 认购金额 >500 万港币 | 大户通道，按比例分配 |
| **甲头** | 甲组中最小认购 | 最受欢迎，中签率最高 |
| **乙头** | 乙组中最小认购 | 资金门槛约 500-600 万 |
| **回拨机制** | 根据认购情况调整甲乙组比例 | 热门股甲组比例提高 |
| **顶头槌** | 最大认购额度 | 甲组顶格或乙组顶格 |

#### 绿鞋机制详解（重要！）

**一句话解释**：绿鞋机制就是给新股上市后的股价买了一份「保险」，防止股价破发跌太惨。

**运作原理**：
```
情况 A：股价上涨 👍
发行价 10 元 → 涨到 12 元
承销商行使绿鞋 → 额外发行 15% 股票
公司多融资 → 三赢！

情况 B：股价下跌 👎
发行价 10 元 → 跌到 8 元（破发）
承销商用超额配售的钱从市场买入 → 托住股价
减少跌幅 → 绿鞋发挥作用
```

**对散户的意义**：
- ✅ 有绿鞋 = 有保险，破发时有人托底
- ✅ 承销商会尽力稳住股价（至少 30 天）
- ⚠️ 但不是 100% 不破发，只是减少跌幅
- ⚠️ 保护期只有 30 天，过期后自生自灭

**实际案例**：
| 新股 | 发行价 | 首日表现 | 绿鞋作用 |
|------|--------|----------|----------|
| 快手 (2021) | 105 元 | +194% | ✅ 行使绿鞋，多融资 80 亿 |
| 哔哩哔哩 (2021) | 808 元 | -2.3% | ✅ 托市成功，减少跌幅 |
| 某地产公司 (2022) | 5.5 元 | -25% | ❌ 绿鞋失效，市场太差 |

**打新建议**：
- 🟢 有绿鞋 + 热门股 → 积极认购
- 🟡 有绿鞋 + 冷门股 → 谨慎参与
- 🔴 没绿鞋 + 冷门股 → 建议回避

> 📖 详细解释见：`GREEN_SHOE_EXPLAINED.md`

|------|--------|----------|------|

1. **换汇**：人民币 → 港币（银行 APP）
2. **汇款**：银行卡 → 证券账户
3. **时间**：通常 1-2 个工作日

**建议金额**：
- 现金打新：至少 5 万港币
- 孖展打新：至少 10 万港币（可杠杆）

### 第三步：开通权限

1. 完成风险测评
2. 签署港股交易协议
3. 开通新股认购权限

## 选股分析

### 基本面分析

**财务指标**：
```
✓ 营收增长率 > 30%
✓ 毛利率 > 40%
✓ 净利润率 > 15%
✓ 资产负债率 < 60%
```

**行业地位**：
- 行业排名前 3
- 市场份额 > 10%
- 有竞争壁垒（技术/品牌/渠道）

**股东背景**：
- 知名机构投资（腾讯、阿里、高瓴等）
- 基石投资者阵容
- 管理层持股情况

### 市场热度分析

**超募倍数**：
- < 20 倍：冷淡
- 20-50 倍：正常
- 50-100 倍：热门
- > 100 倍：非常热门

**认购人数**：
- < 5 万：冷淡
- 5-10 万：正常
- 10-20 万：热门
- > 20 万：非常热门

### 保荐人分析

**历史表现好的保荐人**：
| 保荐人 | 项目数 | 胜率 | 平均回报 |
|--------|--------|------|----------|
| 中金公司 | 50+ | 68% | 45% |
| 中信证券 | 40+ | 65% | 42% |
| 海通国际 | 30+ | 62% | 38% |
| 摩根士丹利 | 20+ | 70% | 50% |

## 申购流程

### 时间线

```
T-7 日：招股开始（周一）
T-4 日：招股截止（周四中午 12:00）
T-3 日：公布中签（周五）
T-2 日：暗盘交易（周一 16:15-18:30）
T-1 日：休息（周二）
T 日：  正式上市（周三 09:30）
```

### 操作步骤

**1. 招股期间**
```
打开券商 APP → 新股页面 → 选择股票 → 点击认购
```

**2. 选择认购数量**
- 现金认购：用自己的钱
- 孖展认购：借钱认购（有利息）

**3. 支付费用**
```
认购费 = 50-100 港币
孖展利息 = 认购金额 × 利率 × 天数/365
交易费 = 0.0027%（交易所收取）
```

**4. 等待结果**
- T-3 日公布中签
- 中签：股票到账
- 未中：资金退回

### 费用明细表

| 费用类型 | 金额 | 说明 |
|----------|------|------|
| 认购费 | 50-100 港币 | 券商收取，无论中签与否 |
| 交易费 | 0.0027% | 港交所收取 |
| 经纪费 | 0.03-0.25% | 券商收取，有最低收费 |
| 孖展利息 | 5-8%/年 | 按实际天数计算 |
| 平台费 | 50-100 港币 | 部分券商收取 |

**示例计算**（10 倍孖展认购 10 万）：
```
本金：10,000 港币
孖展：90,000 港币
认购费：100 港币
孖展利息：90,000 × 6.8% × 7/365 = 117 港币（假设 7 天）
总成本：100 + 117 = 217 港币

如果中签，卖出时还需支付：
交易费 + 经纪费 ≈ 50-100 港币
总成本 ≈ 300 港币

需要涨幅 > 0.3% 才能回本
```

## 风险管理

### 仓位管理

**建议配置**：
```
单只股票不超过总资金的 20%
每次打新用总资金的 30-50%
保留 50% 现金应对机会
```

### 止盈止损

**止盈策略**：
- 暗盘涨 > 30%：卖出一半
- 上市涨 > 50%：全部卖出
- 上市涨 < 10%：观察持有

**止损策略**：
- 暗盘跌 > 10%：考虑卖出
- 上市跌 > 20%：坚决止损
- 破发股：不要补仓

### 常见陷阱

❌ **All-in 单只股票** → 分散投资
❌ **盲目追高** → 理性分析
❌ **破发补仓** → 及时止损
❌ **借钱打新** → 量力而行

## 常见问题

### Q1: 新手适合打新吗？

**适合**，但要注意：
- 从小额开始（1-2 万）
- 选择稳健股票
- 学习基础知识

**新手建议路线**：
```
第 1 个月：学习 + 观察（不操作）
第 2 个月：小额试水（1-2 万/只）
第 3 个月：形成策略（3-5 万/只）
第 4 个月：稳定盈利（5-10 万/只）
```

### Q2: 需要多少资金？

**建议**：
- 现金打新：5 万起
- 孖展打新：10 万起（可用 10 倍杠杆）
- 专业玩家：50 万+

```
总资金 10 万：
- 5 万用于现金打新（50%）
- 3 万用于孖展打新（30%）
- 2 万现金备用（20%）

```

```
冷门股：
- 用孖展提高认购数量
- 乙头中签率更高
```

### Q4: 破发了怎么办？

**应对策略**：
1. 设置止损线（-20%）
2. 不要补仓摊平
3. 及时卖出止损
4. 总结经验教训

**止损决策树**：
```
破发 → 看暗盘表现
  │
  ├─ 暗盘跌 <10% → 持有，等上市
  │   │
  │   └─ 上市涨 → 卖出止盈
  │   └─ 上市跌 → 止损卖出
  │
  └─ 暗盘跌 >10% → 立即卖出
      │
      └─ 减少损失，保住本金
```

### Q5: 怎么选股？

**五步选股法**：
```
第 1 步：看行业
  • 是否热门行业？
  • 有政策支持吗？
  
第 2 步：看基本面
  • 营收增长如何？
  • 毛利率多少？
  
第 3 步：看热度
  • 超募倍数多少？
  • 认购人数多少？
  
第 4 步：看保荐人
  • 历史胜率如何？
  • 近期表现怎样？
  
第 5 步：看绿鞋
  • 有绿鞋机制吗？
  • 保护期多久？
```

**快速筛选表**：
| 条件 | 符合 | 不符合 |
|------|------|--------|
| 热门行业 | ✅ 加分 | ❌ 减分 |
| 营收增长>30% | ✅ 加分 | ❌ 减分 |
| 超募>50 倍 | ✅ 加分 | ❌ 减分 |
| 好保荐人 | ✅ 加分 | ❌ 减分 |
| 有绿鞋 | ✅ 加分 | ❌ 减分 |

5 个都符合 → 积极认购
3-4 个符合 → 谨慎参与
0-2 个符合 → 建议回避

### Q6: 什么时候卖出？

**卖出时机选择**：

| 情况 | 建议 | 原因 |
|------|------|------|
| 暗盘涨>50% | 全部卖出 | 落袋为安 |
| 暗盘涨 20-50% | 卖出一半 | 锁定部分收益 |
| 暗盘涨<20% | 持有 | 等上市 |
| 暗盘跌<10% | 持有 | 等反弹 |
| 暗盘跌>10% | 立即卖出 | 止损 |

**上市首日卖出策略**：
```
开盘涨>30% → 立即卖出（止盈）
开盘涨 10-30% → 分批卖出
开盘涨<10% → 观察持有
开盘跌<10% → 考虑止损
开盘跌>10% → 坚决止损
```

### Q7: 孖展打新安全吗？

**孖展风险**：
- ✅ 好处：提高中签率，放大收益
- ❌ 风险：放大亏损，要付利息

**安全使用孖展**：
```
1. 只用闲钱打新
2. 孖展倍数不超过 10 倍
3. 选择热门股用孖展
4. 冷门股不要上孖展
5. 设置止损线
```

**孖展成本计算**：
```
10 万本金，10 倍孖展：
- 自有资金：10 万
- 借款：90 万
- 总认购：100 万

成本：
- 认购费：100 元
- 利息：90 万×6.8%×7/365=1174 元
- 总成本：1274 元

需要涨幅>0.13% 回本
```

**建议**：
- 🟢 热门股：可以用 10 倍孖展
- 🟡 一般股：5 倍孖展或不用
- 🔴 冷门股：不要用孖展
