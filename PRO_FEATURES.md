# 🦞 港股打新助手 - 专业版功能说明

## ✅ 四大核心模块

### 📅 模块一：新股日历

**功能**：
- 🔥 **正在招股**：显示当前可认购的新股
- 📅 **即将上市**：显示已公布上市日期的新股
- 📊 **全部新股**：完整的新股列表表格

**数据字段**：
| 字段 | 说明 |
|------|------|
| 代码 | 港股代码（5 位数字） |
| 名称 | 公司全称 |
| 招股价 | 发行价格范围 (HK$) |
| 招股日期 | 认购起止日期 |
| 上市日期 | 预计上市时间 |
| 状态 | 正在招股/待上市/已上市 |

**统计卡片**：
- 正在招股数量
- 即将上市数量
- 本月新股数量
- 港股 IPO 总数

---

### 📊 模块二：基本面分析

**公司概况**：
- 行业分类
- 业务模式
- 营收结构
- 市场地位

**估值参考**：
- 行业平均 PE
- 行业平均 PB
- 新股平均涨幅
- 可比公司对比

**保荐人历史表现**：
- 今年项目数量
- 历史胜率
- 平均回报率
- Top 保荐人排名

**打新策略建议**：
| 策略类型 | 适合资金 | 特点 |
|----------|----------|------|
| 💎 甲头策略 | <10 万 | 一手必中，适合散户 |
| 💰 乙头策略 | 10-50 万 | 提高中签率 |
| 🚀 融资策略 | >50 万 | 10 倍孖展，最大化收益 |

### 🤖 模块三：AI 打新策略

**AI 综合推荐**：
- 🔥 **建议认购**：高置信度推荐（绿色）
- ⏳ **谨慎观察**：中等置信度（橙色）
- ❌ **不建议**：低置信度（红色）

**市场情绪指标**：
- 超募倍数（热度）
- 认购人数（参与度）
- 暗盘胜率（历史表现）

**风险提示**：
- ⚠️ 估值偏高
- 📉 冷门行业
- 🔴 市场波动
- 💸 流动性风险

**参考依据**：
1. 历史新股表现数据（近 3 个月）
2. 保荐人历史胜率统计
3. 行业估值对比分析
4. 市场情绪指标
5. 基石投资者阵容
6. 公司基本面分析

### 📚 模块四：打新课程

**什么是港股打新**：
- 核心逻辑
- 盈利模式
- 风险提示

**甲组 vs 乙组**：
| 对比项 | 甲组（小户） | 乙组（大户） |
|--------|-------------|-------------|
| 认购金额 | <500 万港币 | >500 万港币 |
| 分配机制 | 一手必中 | 按比例分配 |
| 适合人群 | 散户 | 机构 |
| 资金门槛 | 低 | 高 |

**孖展认购流程**：
1. 选择孖展倍数（5x/10x/20x）
2. 支付首付（10%-20%）
3. 支付利息（5%-8%/年）
4. 等待结果

**常见术语**：
- 一手、入场费、中签率
- 超募倍数、暗盘、孖展
- 基石投资者、绿鞋机制

## 🔧 故障排查

### 网页无法加载

1. **检查 API 服务**：
```bash
ps aux | grep hk_ipo_api
```

2. **重启服务**：
```bash
bash ~/.openclaw/skills/scrapling/scripts/stop_hk_ipo_server.sh
bash ~/.openclaw/skills/scrapling/scripts/start_hk_ipo_server.sh
```

3. **查看日志**：
```bash
tail -50 /tmp/hk_ipo_api.log
```

### 数据不显示

1. **打开浏览器控制台**（F12）
2. **查看错误信息**
3. **测试 API**：http://localhost:8765/api/latest
